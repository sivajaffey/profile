What is SampleJS ?

SampleJS is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling. Further Updates in progress..

Use this CDN link on Your Header in your Html page and get regular updates. {{cdnLink}}
s(document).ready()
s(document).ready(function());

This to get and update value after document is loaded.

s(type.class#id).htmlValue()
s('p#id.class').htmlValue();

This attributes is used to get the value of the html file like p, a and other tag's s(type.class#id).htmlValue(value) -- set value; s(type.class#id).htmlValue() -- get value;

s(type.class#id).value()
s('p#id.class').htmlValue();

This to get value from input s(type.class#id).value(value) -- to update or set value in input s(type.class#id).value() -- to update or get value form input

s(type.class#id).next()
s('p#id.class').htmlValue();

This is used to set value in input s(type.class#id).next() -- to get next value in input s(type.class#id).next(value) -- to update or set value in next input

s(type.class#id).prev()
s('p#id.class').htmlValue();

s(type.class#id).prev() -- to get previous value in input s(type.class#id).prev(value) -- to update or set value in previous input

s(type.class#id).on()
s(type.class#id).on('click',function());

s(type.class#id).on('click',function()) -- this on function supports all on functions like click, keypress, keyup etc.
